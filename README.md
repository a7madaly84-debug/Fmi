# DawaaConnect - دليل الرفع اليدوي (Netlify Drop)

لرفع المشروع يدوياً على Netlify، اتبع الخطوات التالية:

## 1. البناء المحلي (Local Build)
افتح التيرمينال في مجلد المشروع وقم بتنفيذ الأوامر التالية:
```bash
npm install
npm run build
```
سيظهر لك مجلد جديد باسم `dist`. هذا المجلد يحتوي على كل ملفات التطبيق الجاهزة.

## 2. الرفع على Netlify
1. اذهب إلى [Netlify Drop](https://app.netlify.com/drop).
2. قم بسحب وإفلات مجلد `dist` داخل الصفحة.
3. انتظر حتى ينتهي الرفع.

## 3. ضبط مفتاح Gemini API
بعد الرفع، يجب إضافة مفتاح الـ API ليعمل الذكاء الاصطناعي:
1. اذهب إلى إعدادات الموقع في Netlify.
2. اختر **Site configuration** > **Environment variables**.
3. أضف متغيراً جديداً:
   - Key: `API_KEY`
   - Value: (ضع مفتاح Gemini الخاص بك هنا)
4. قم بإعادة النشر (Trigger deploy) لتطبيق التغييرات.

---
**ملاحظة:** تم ضبط ملف `netlify.toml` لضمان عمل الروابط بشكل صحيح عند تحديث الصفحة (Refresh).
